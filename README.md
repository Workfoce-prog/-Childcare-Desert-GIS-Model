# Childcare Desert GIS Model

**Best-fit division:** Assistant Commissioner: Early Childhood / Child Care Services Division

## Purpose
Identifies ZIP codes/counties where childcare supply is limited relative to need. Production version can connect to tract shapefiles, provider licensing records, and GIS coordinates.

## Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Presenting language
This is a transparent decision-support prototype. It does not replace eligibility decisions, caseworker judgment, or statutory requirements. It helps identify where prevention, resources, and coordinated supports may be needed earlier.
